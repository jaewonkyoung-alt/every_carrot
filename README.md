# 에브리당근 (임시명)

> 캠퍼스 전용 중고 물품 거래 서비스

우리 학교 학생들끼리 안전하게 중고 물품을 사고팔 수 있는 웹 서비스입니다.
학교 이메일 인증을 통해 재학생만 이용할 수 있으며, 실시간 채팅으로 편리하게 거래할 수 있습니다.

---

## 기술 스택

| 영역 | 기술 |
|------|------|
| **프론트엔드** | React (Vite), TailwindCSS, React Router, Zustand, Axios |
| **백엔드** | Node.js (Express), MySQL 8.0, JWT, Multer |
| **실시간 통신** | Socket.io |
| **협업 도구** | Git/GitHub, Postman, Figma |

---

---

## 주요 기능

- 학교 이메일 인증 기반 회원가입/로그인
- 중고 물품 등록/수정/삭제 (사진 업로드)
- 카테고리별 필터 + 키워드 검색
- 거래 상태 관리 (판매중/예약중/판매완료)
- 판매자-구매자 1:1 실시간 채팅
- PC/모바일 반응형 웹

## 실행 방법

### 사전 준비
- Node.js v20 LTS
- MySQL 8.0 + `campus_market` DB 생성

### 백엔드
```bash
cd server
cp .env.example .env   # 환경변수 설정
npm install
npm run dev            # http://localhost:4000
```

### 프론트엔드
```bash
cd client
npm install
npm run dev            # http://localhost:5173
```
